NiPyPE¶

NiPyPE is a Python library that interfaces to existing neuroimaging software such as SPM, FSL and Freesurfer. It also features a workflow engine that enables it to distribute jobs over multiple machines or clusters. It is able to pull data from XNAT and push back the results using pyxnat.
